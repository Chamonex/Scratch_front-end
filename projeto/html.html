HTML

Quando lemos um texto, conseguimos compreender o que é o título, o que é o parágrafo, uma lista, etc...

Entretanto, o nosso navegador (Chrome, Opera, Edge...) precisa da indicação do que é cada elemento do site que estamos fazendo.

Para isso serve o HTML !

  • Marcar os elementos (título, parágrafo, imagem, etc)
  • Apresentar o conteúdo (texto do título, do parágrafo, o arquivo da imágem, etc)

por que temos que criar o nosso html principal com o nome "index.html" ?

Fazemos isso porque quando fizermos o upload de todos os arquivos do nosso site em um servidor, não precisaremos indicar qual página é para abrir primeiro.

Imagine que upamos uma pasta "LOJA" para um server com os seguintes arquivos:

  • home.html
  • produtos.html
  • contato.html
  • style.css
  • script.js

Quando acessarmos esse site, o servidor ficará em dúvida em qual página abrir primeiro (home, produtos ou contato).

Como queremos abrir a página home primeiro, nomeamos ela como "index.html", e o nosso navegador irá saber que ela deverá ser carregada.

HTML TAGs

O HTML utiliza as TAGS para marcar os elementos (título, imagem, parágrafo, link...) e indicar para o nosso anvegador o que é o que.

As tags ficam entre os símbolos de menor que e maior que.

Exemplos de tags:

  • p criar um parágrafo
  • h1 criar um título principal
  • h2 criar um subtítulo
  • br pular uma linha em branco
